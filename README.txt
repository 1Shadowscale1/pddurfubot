Данилов Илья и Владынцев Сергей

Бот связанный с ПДД, решение экзаменов по ПДД, показ различных статей из ПДД и т.д.

0 таск (готово) - help, возвращает доступные команды бота на данный момент, также about - возвращает имена разработчиков (нас самих)
1 таск (в разработке) - диалоги (прохождение юзером 20 вопросов ПДД и дальнейший вывод сдан ли экзамен)
2 таск - база данных (показ статей ПДД по разделам и номерам, парсинг и добавление новых с сайта)
3 таск - реализация использования клавиатур для первых двух тасков (?)
4 таск - отправление сообщений ботом с прикрепленными картинками
